## What it is

**AgentMemory** is the #1 persistent memory solution for AI coding agents, based on real-world benchmarks. It gives coding agents (Claude Code, Codex, Cursor, Copilot) the ability to remember context across sessions — files, decisions, patterns, and user preferences. Source on [GitHub](https://github.com/rohitg00/agentmemory). ~25k stars, Apache-2.0, TypeScript.

## Best for

- **AI agents that need to remember across sessions** — install AgentMemory and your coding agent retains context (files touched, decisions made, patterns learned) between conversations.
- **Teams using Claude Code / Cursor / Codex** — works with the major coding agent harnesses.

## Not for

- Agents that don't support plugins/skills.
- Non-coding AI agents (it's specifically for coding-agent memory).
