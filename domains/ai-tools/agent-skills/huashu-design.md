## What it is

**Huashu Design** is an HTML-native design skill for Claude Code. It enables high-fidelity prototypes, slides, and animations directly in HTML, with 20 design philosophies, a 5-dimension review system, and MP4 export. Agent-agnostic (works with any AI coding agent). ~21k stars, MIT, HTML.

## Best for

- **AI agents building high-fidelity HTML prototypes** — install the skill and the agent produces polished, native-HTML designs (not frameworks, not AI slop).
- **Creating slides + animations in HTML** — the skill includes slide + animation capabilities with MP4 export.
- **Design review** — 5-dimension review system + 20 design philosophies guide the agent.

## Not for

- Non-HTML design (it's HTML-native specifically).
- Agents that don't support skills.
