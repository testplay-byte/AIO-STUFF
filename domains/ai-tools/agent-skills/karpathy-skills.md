## What it is

**Andrej Karpathy Skills** is a single `CLAUDE.md` file that improves Claude Code behavior, derived from Andrej Karpathy's observations on LLM coding pitfalls. It's a behavior-modification skill — drop it into your project and Claude Code avoids common LLM coding mistakes. ~191k stars, no license, by multica-ai.

## Best for

- **AI agents (and their operators) that want better coding behavior** — drop the CLAUDE.md into your project and the agent follows Karpathy's observations on what LLMs get wrong.
- **Claude Code workflows** — designed specifically for CLAUDE.md but the principles apply to any coding agent.

## Not for

- Agents that don't read CLAUDE.md or equivalent config.
- Commercial use without verifying the license (no license file — treat as all-rights-reserved).
