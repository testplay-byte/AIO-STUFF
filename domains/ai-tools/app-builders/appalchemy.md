## What it is

**AppAlchemy** is a mobile app builder — build React Native apps in your browser and preview
them live on your phone. Hosted at [appalchemy.ai](https://appalchemy.ai). Proprietary.

## Best for

- **Users building React Native apps visually** — design + build in the browser, preview live
  on your phone.
- **Rapid mobile app prototyping** — no local dev setup needed.

## Not for

- AI agents looking for an API (browser-only; AI compatibility: 2/5).
- Non-mobile apps (it's React Native specifically).
