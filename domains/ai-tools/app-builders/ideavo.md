## What it is

**Ideavo** is a free AI app builder — build production-ready websites, mobile apps, games,
and internal tools by chatting with AI. Bring your own model API key (OpenAI, Anthropic,
Google, or any provider — no markup), describe the app, watch it generate auth routes,
database schemas, middleware, APIs, and business logic, then export to GitHub and deploy to
Vercel. Hosted at [ideavo.ai](https://ideavo.ai). Freemium SaaS ($0 Free / ~$5 / ~$10
tiers; "Agent+" is the paid tier). SOC 2 certified. Used by people at Amazon, Microsoft,
LinkedIn, Google, Adobe, Notion, Slack, Spotify, and more.

## Best for

- **AI agents (and humans) going from idea to shipped app by chatting** — describe the app,
  Ideavo generates the backend (auth, DB, payments, business logic) + frontend, exports to
  GitHub, deploys to Vercel. Good for MVPs, internal tools, rapid prototyping.
- **Teams that already have model API keys** — connect your existing OpenAI/Anthropic/Google
  key with no markup; you pay the model provider directly.
- **Multi-platform shipping** — one codebase → web (Next.js) + mobile (Expo), zero
  configuration.

## Not for

- Teams that want a fully managed model (no BYOK) — Ideavo requires you to bring your own
  API key or subscription.
- Self-hosted / offline workflows (it's a cloud SaaS — no local install, no source).
- Fine-grained manual control over every line (it's AI-generated; you steer via chat, not
  by writing code directly — though you can export and edit the generated code).

## How it works

Ideavo runs in the browser. You connect your own model API key (OpenAI/Anthropic/Google/any),
then describe the app you want in chat. The AI generates the backend (auth routes, DB schema,
permissions middleware, product APIs, payments) and frontend (Next.js for web, Expo for
mobile) in real time, showing each step as it builds. When satisfied, export the codebase to
GitHub and deploy to Vercel from the same workflow. No local install — it's a cloud SaaS.

Not applicable — this is a web SaaS, not a CLI/library. To use:
