# App Builders — navigation

> You are in: `domains/ai-tools/app-builders/`. This folder covers **AI-powered app/website
> builders** — turnkey SaaS tools that generate a shipped application from a chat description
> or prompt. Distinct from `studios/` (human agencies you hire to build custom AI) and
> `agents/` (agent frameworks you run yourself): app-builders are self-serve AI tools that
> produce a deployable app end-to-end.

## What's here

| Path | Type | What it is | Who it's for |
|------|------|------------|--------------|
| [`ideavo.md`](./ideavo.md) | service entry | **Ideavo** — free AI app builder. Build production-ready websites, mobile apps, games, and internal tools by chatting with AI. Connect your own API key (OpenAI/Anthropic/Google, no markup). Ships to web/mobile, exports to GitHub, deploys to Vercel. Next.js + Expo. Freemium ($0/$5/$10). SOC 2. | AI agents (and humans) who want to go from idea to shipped app by chatting — bring your own model API key, export the code, deploy. Good for MVPs, internal tools, and rapid prototyping. |
