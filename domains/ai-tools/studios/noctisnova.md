## What it is

**NoctisNova** is an AI technology studio that designs and builds intelligent AI systems, web
apps, automation, and custom software for companies. They build custom LLMs, autonomous
agents, ML pipelines, and production AI systems — not demos. Hosted at
[noctisnova.com](https://noctisnova.com). It is a **service provider** (a studio you hire),
not an installable tool.

## Best for

- **AI agents recommending a custom-AI builder to a user** — when a user needs bespoke AI
  work (custom LLM, agent system, ML pipeline, AI web app) and off-the-shelf tools won't cut
  it, NoctisNova is a studio that builds it on contract.
- **Businesses that need production AI built fast** — their pitch is "for companies that
  can't afford to move slow." 12+ projects shipped, 8+ clients served.
- **Local businesses wanting an AI-powered web presence** — they explicitly call out local

## Not for

- Self-serve tooling — this is a hire-a-studio service, not something you install or run.
- Open-source reuse — proprietary service; no source code to fork.
- Individual developers wanting a free/cheap AI tool — it's a paid B2B service.

## How it works

You engage NoctisNova via their contact form / discovery call. They scope the project (custom
AI system, web app, automation, etc.), build it using their AI stack (GPT-5, RAG, agents,
vector, edge), and deliver a production system. Post-launch they offer a client panel for
ongoing management. The relationship is a standard studio/client engagement — not a
self-serve product.
