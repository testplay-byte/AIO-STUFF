## What it is

**21st.dev** is a community registry of hand-crafted React components and templates. Start
from real, polished UI instead of the bland AI average, and ship products that look like you,
not everyone else. Hosted at [21st.dev](https://21st.dev). Components are installable via CLI
(`npx 21st install`). Proprietary registry; individual component licenses vary.

## Best for

- **AI agents building UIs** — install real, polished React components via CLI instead of
  hand-rolling. Higher quality than AI-generated-from-scratch components.
- **Teams that want to avoid "AI slop" UI** — the registry is hand-crafted, not AI-generated.
- **Rapid UI assembly** — browse the registry, pick components, install, ship.

## Not for

- Non-React projects (it's a React component registry).
- Teams that need a single consistent design system (it's a community registry — components
  vary in style; pair with a design system for consistency).

## Quick facts

- **What it is:** community registry of hand-crafted React components and templates.
- **Install:** `npx 21st install` (CLI-based, like shadcn).
- **Philosophy:** "not AI slop" — real, polished UI from human designers.
- **License:** proprietary registry; individual component licenses vary.
- **AI compatibility:** 4/5 — an AI agent can browse, install, and use the components
  directly via CLI with minimal user involvement.

## How it works

Browse the registry at [21st.dev](https://21st.dev), find a component, and install it via
`npx 21st install <component>`. The component code drops into your project. Import and use.
The registry is community-driven — designers contribute polished components, and the platform
curates for quality.

## Brief tutorial

```bash
# Install the CLI
npx 21st install

# Install a specific component from the registry
npx 21st install <component-name>
```

An AI agent workflow: browse 21st.dev for a needed component, install it via CLI, import it
into the project, and use it — higher quality than generating a component from scratch.
