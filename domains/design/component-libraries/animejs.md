## What it is

**Anime.js** is a lightweight JavaScript animation engine. Animate CSS properties, SVG, DOM
attributes, and JavaScript objects with a simple, powerful API. ~71k stars, MIT, JavaScript.
By Julian Garnier. Hosted at [animejs.com](https://animejs.com).

## Best for

- **AI agents adding animation to a web project** — install anime.js, animate any CSS
  property, SVG, or DOM element with a clean API.
- **Developers wanting a lightweight, powerful animation engine** — no dependencies, 17kb
  minified.

## Not for

- Non-JS projects.
- CSS-only animations that don't need JS control (use CSS transitions/animations instead).
