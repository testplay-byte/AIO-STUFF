## What it is

**Smooothy** is a smooth, configurable, extendable slider library made for animation. It provides high-quality slider components with smooth transitions. Hosted at [smooothy.federic.ooo](https://smooothy.federic.ooo). By federic.

## Best for

- **AI agents adding sliders to a project** — install the library and use its configurable API.
- **Developers wanting smooth, animation-focused sliders** — built specifically for animation quality.

## Not for

- Non-JS projects.
- Full UI component systems (it's a slider library specifically).
