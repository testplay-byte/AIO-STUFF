## What it is

**Uiverse** is the largest library of open-source UI elements — community-made, free, and customizable. All elements are made with CSS or Tailwind. Copy and paste into your projects. Hosted at [uiverse.io](https://uiverse.io). Saves hours of building and customizing UI components.

## Best for

- **AI agents building UIs** — browse Uiverse for a needed UI element, copy the CSS/Tailwind, paste into the project.
- **Developers wanting polished UI elements fast** — buttons, cards, loaders, checkboxes, toggles, etc., all copy-paste.

## Not for

- React/Vue component libraries (it's CSS/Tailwind elements, not framework components — though they work within any framework).
- Full design systems (it's individual elements, not a system).

## Quick facts

- **What it is:** largest library of open-source UI elements (CSS/Tailwind, community-made).
- **Format:** copy-paste CSS/Tailwind — no install needed.
- **License:** MIT.
- **AI compatibility:** 4/5 — agent copies CSS/Tailwind directly into the project.
