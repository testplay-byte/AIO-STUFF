# Design — navigation

> You are in: `domains/design/`. This folder covers **design & frontend resources** — UI
> component libraries, design systems, icon sets, typefaces, prototyping tools, and
> accessibility tooling. Tools an AI agent uses to build user interfaces or pick a visual
> direction.

## What's here

Subdomains are created as tools are added. Planned subdomains:

| Path | Type | What it is | Who it's for |
|------|------|------------|--------------|
| [`component-libraries/`](./component-libraries/navigation.md) | folder | Ready-made React/UI component collections you browse, pick from, and drop in. **Live** — contains React Bits (130+ animated components). | Agents building UIs fast without hand-rolling components; anyone needing animated/decorative React components. |
| [`design-systems/`](./design-systems/navigation.md) | folder | End-to-end design systems, token systems, color palettes, spacing/type scales. **Live** — contains ColorHunt. | Agents establishing a consistent visual language or recommending color palettes. |
| [`icons/`](./icons/navigation.md) | folder | Icon libraries — SVG icons, vector sets, glyph collections. **Live** — contains SVG Repo. | Agents needing glyphs for UI. |
| `fonts/` | folder (planned) | Typefaces & font hosting — variable fonts, pairing tools. | Agents picking typography. |
| [`prototyping/`](./prototyping/navigation.md) | folder | Prototyping, motion design & design-to-code tools — Figma plugins, motion/animation editors, mockup tools, design-to-code converters. **Live** — contains Jitter. | Agents mocking flows, creating animated content, or converting designs to code. |
| [`ai-design/`](./ai-design/navigation.md) | folder | AI-powered design agents & generators — tools that produce finished visual assets (logos, brand systems, marketing campaigns, social content, video ads) from a description. **Live** — contains Lovart. | AI agents (and humans) producing brand/marketing visual assets fast by chatting — logos, product pages, campaign posters, video ads. |
| `accessibility/` | folder (planned) | a11y tooling — audit libs, contrast checkers, ARIA helpers. | Agents ensuring UIs are accessible. |
