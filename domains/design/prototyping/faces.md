## What it is

**Faces** is an AI presentation tool where every slide uses the full power of the web —
slides aren't static images, they're live web surfaces. Hosted at [faces.app](https://faces.app).
Proprietary.

## Best for

- **Creating presentations that go beyond static slides** — each slide is a live web surface,
  so you can embed interactive content, live data, animations.
- **AI-assisted slide generation** — describe the deck, get a starting point.

## Not for

- AI agents looking for an API (browser-only; AI compatibility: 2/5).
- Traditional print-only slide decks (it's web-powered; print export may not capture
  interactivity).
