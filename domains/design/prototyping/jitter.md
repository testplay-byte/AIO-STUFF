## What it is

**Jitter** is a collaborative, web-based motion design tool that lets you create professional
animations in minutes — no motion-design experience required. Import from Figma, export to
4K video / GIF / Lottie, with AI agents ("Superagents") built in to speed up creation.
Hosted at [jitter.video](https://jitter.video). 20,000+ creative teams use it (Perplexity,
Deliveroo, 27b among the named customers). It is a **freemium SaaS service** (not open
source, not installable).

## Best for

- **AI agents (and their operators) creating animated content fast** — promo reels, social
  clips, UI motion, motion graphics. The built-in AI agents ("Superagents") accelerate the
  from-idea-to-motion step.
- **Teams that design in Figma and need to animate** — direct Figma import means no
  re-creating assets; animate the existing frames and export as video/GIF/Lottie.
- **Shipping motion to product** — Lottie export drops straight into web/mobile apps; 4K
  export for marketing channels.

## Not for

- Self-hosted / offline workflows (it's a cloud SaaS — no local install, no source).
- Free unlimited use (free tier exists but paid tiers for Pro/Team/Enterprise features).
- Open-source reuse (proprietary — you can't fork or audit the source).

## How it works

Jitter runs entirely in the browser. You start from a template or import a Figma frame, then
animate via a timeline-based editor (keyframes, effects, shaders). The built-in AI agents
("Superagents") can generate or accelerate motion from a prompt. Collaborators can join the
same project in real time. Export targets cover marketing (4K video, GIF) and product
(Lottie for web/mobile). No local install — it's a cloud SaaS.

## Brief tutorial

Not applicable — this is a web SaaS, not a CLI/library. To use:

1. Visit [jitter.video](https://jitter.video) and sign up (free tier available).
2. Start from a template or import a Figma frame.
3. Animate via the timeline editor, or use the AI "Superagents" to generate motion.
4. Export as 4K video, GIF, or Lottie.

clip / motion graphic / UI motion, fast, and I design in Figma."
