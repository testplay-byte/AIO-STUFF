## What it is

**Pretext** is a fast, accurate, and comprehensive text measurement and layout library. It handles text measurement (width, height, line breaks) and layout (positioning, wrapping, alignment) with high precision. Source on [GitHub](https://github.com/chenglou/pretext). ~49k stars, MIT, TypeScript. By chenglou (the original author of ReasonML/Roc).

## Best for

- **AI agents building text-heavy UIs** — install Pretext for accurate text measurement + layout.
- **Projects needing precise text rendering** — fast + comprehensive measurement and layout.

## Not for

- Non-text layout (it's specifically for text measurement + layout).
- Environments where CSS already handles text layout adequately.
