## What it is

**Exercism** is a free platform to get really good at programming. Develop fluency in 83
programming languages via a unique blend of learning, practice, and mentoring. 8,500+
coding exercises. An independent, community-funded, not-for-profit organization — 100% free,
forever. Hosted at [exercism.org](https://exercism.org).

## Best for

- **Learning a new programming language** — 83 language tracks (Python, JavaScript, Java, C++,
  C#, Go, C, Rust, TypeScript, Bash, PHP, Ruby, and 71 more) with progressive exercises.
- **Practicing coding** — 8,500+ exercises across all tracks, with automated feedback.
- **Mentoring** — free human mentoring on exercises (a unique offering).

## Not for

- AI agents looking for an API to call (it's a browser learning platform; AI compatibility:
  2/5 — an agent can recommend it to a user but doesn't use it directly).
- Competitive coding / interview prep specifically (it's learning-focused, not
  interview-gaming; though it helps with fundamentals).
- Teams looking for a paid enterprise training platform (it's free/community).

## Quick facts

- **Languages:** 83 programming language tracks.
- **Exercises:** 8,500+ across all tracks.
- **Mentoring:** free, human, volunteer-based.
- **Cost:** 100% free, forever. Not-for-profit, community-funded.
- **Popular tracks:** Python (761k+ students), JavaScript (480k+), Java (266k+), C++
  (187k+), C# (174k+), Go (160k+), C (149k+), Rust (136k+), TypeScript (93k+), Bash (82k+).
- **AI compatibility:** 2/5 — user-facing learning platform; an AI agent can recommend it
  but doesn't use it directly.

## How it works

Sign up free at [exercism.org](https://exercism.org), pick a language track, and work
through exercises progressively. Each exercise comes with automated feedback; some tracks
offer free human mentoring. The platform is browser-based with a CLI tool for local
practice.
