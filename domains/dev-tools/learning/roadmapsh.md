## What it is

**roadmap.sh** is a community-driven platform with developer roadmaps, articles, and guides
for developers to grow in their career. Step-by-step learning paths for frontend, backend,
DevOps, AI, mobile, and more. Hosted at [roadmap.sh](https://roadmap.sh). Free.

## Best for

- **Developers planning their learning path** — follow a step-by-step roadmap for a specific
  role (frontend, backend, DevOps, AI, etc.).
- **AI agents recommending learning paths** — consult roadmap.sh to suggest what a user should
  learn next for a given career direction.

## Not for

- AI agents looking for an API (browser-based; AI compatibility: 3/5 — consult as reference).
- Non-developer learning (it's dev-focused).
