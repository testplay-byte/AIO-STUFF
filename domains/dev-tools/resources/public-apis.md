## What it is

**Public APIs** is a collective list of free APIs — one of the most-stard repos on GitHub (~450k stars). A curated catalog of hundreds of free APIs across categories: animals, anime, anti-malware, art & design, books, business, calendar, cloud, cryptocurrency, data, development, dictionaries, documents, environment, events, finance, food, games, government, health, jobs, music, news, open data, open source, patent, personality, phone, photography, programming, science, security, social, sports, test data, text, tracking, transportation, URL shorteners, vehicle, weather. MIT, Python (the list itself is a Python file).

## Best for

- **AI agents finding a free API for a given need** — consult the list by category before building a custom integration.
- **Developers prototyping** — find a free API for data, weather, finance, news, etc.

## Not for

- Paid/premium APIs (it's free APIs only).
- API testing tools (it's a list, not a tester).
