## What it is

**AI Knowledge Cloud** (aikc.vn) is a tools discovery platform for study and work — "find the perfect work & study tools for you." Curated by VinUniversity library, it helps users discover the best tools to build faster and more efficiently. Hosted at [aikc.vn/en](https://aikc.vn/en).

## Best for

- **Students and professionals discovering tools for study or work** — a curated catalog of tools organized by use case.
- **VinUniversity community** — built specifically for their library + knowledge cloud.

## Not for

- AI agents looking for an API (it's a browser platform; AI compatibility: 2/5).
- Non-study/work tool discovery (it's scoped to productivity tools, not dev/security/etc.).
